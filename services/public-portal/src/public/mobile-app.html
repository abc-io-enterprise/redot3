🛡️ Owner APK — Cellular Failsafe Gateway

The owner-only Android APK turns a phone into an autonomous cellular backup gateway for the ABC-IO mesh. It includes biometric authentication and hardcoded failsafe routes to the production VPS.

Owner use only. This APK requires fingerprint or face unlock to operate and is tied to the redot1 / ABC-IO owner account.

Download Owner APK Download Signature

redot2-operator.apk • 45 KB • signed with META-INF/ABCIO.RSA • version 2.0.0

Installation Steps

  1. Download redot2-operator.apk to your Android device.
  2. Go to Settings → Security → Install unknown apps and allow your browser or file manager.
  3. Open the APK file and tap Install.
  4. Launch ABC-IO Gateway and authenticate with biometrics.
  5. Confirm the dashboard shows Primary, AI1, and AI2 nodes as online.
