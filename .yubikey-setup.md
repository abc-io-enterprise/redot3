## Purpose

This file documents the YubiKey hardening procedure for the ABC-IO private administration system. The YubiKey is used to:

1. Sign deployment commits and releases
2. Authenticate SSH access to VPS nodes
3. Decrypt the offline admin archive
4. Provide second-factor authentication for the owner dashboard

## Required YubiKey Configuration

### 1. PIV / OpenPGP (for signing and encryption)

```bash
# Generate OpenPGP key on YubiKey
gpg --card-edit
admin       # enable admin commands
key-attr    # set RSA 4096 or ECC P-256
generate    # generate key on card
name        # set card holder name: Christopher Porreca
login       # set login email: cporreca@abc-io.com
quit
```

### 2. FIDO2 / WebAuthn (for dashboard login)

Register the YubiKey as a WebAuthn credential in the owner dashboard. The backend endpoint is:

```
POST /api/auth/yubikey/register
POST /api/auth/yubikey/verify
```

### 3. SSH (for VPS access)

```bash
# Generate resident SSH key on YubiKey
ssh-keygen -t ed25519-sk -O resident -O verify-required -f ~/.ssh/id_ed25519_sk_abcio

# Copy public key to VPS
ssh-copy-id -i ~/.ssh/id_ed25519_sk_abcio.pub root@162.254.32.142
```

## Enroll This YubiKey

After receiving the physical YubiKey, run:

```bash
./scripts/setup-yubikey.sh
```

This script will:
1. Detect the inserted YubiKey
2. Configure OpenPGP and SSH resident keys
3. Add the SSH public key to all VPS nodes
4. Update the `OWNER_BIOMETRIC_SECRET` derivation to include YubiKey attestation
5. Create `admin-desktop/yubikey-pubkey.txt` for backup

## Key Files

| File | Purpose |
|------|---------|
| `admin-desktop/yubikey-pubkey.txt` | Public SSH key for deployments |
| `admin-desktop/yubikey-gpg.asc` | GPG public key for commit signing |
| `.env` (not committed) | `OWNER_BIOMETRIC_SECRET`, `YUBIKEY_ENABLED=true` |

## Failsafe Access

If the YubiKey is lost, the owner dashboard can still be accessed using the static owner password and biometric token derived from `OWNER_BIOMETRIC_SECRET`. The YubiKey adds a second factor but is not the sole factor.
